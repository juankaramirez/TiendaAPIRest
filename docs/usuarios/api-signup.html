POST api/signup

Permite registrar un nuevo usuario.

URL del recurso http://tiendaapirest.aws.af.cm/api/signup
Parámetros

Requeridos:
nombre=[string]
Ejemplo: nombre = James

apellido=[string]
Ejemplo: apellido = Ramirez

username=[string]
Ejemplo: username = j.ramirez

email=[string@string]
Ejemplo: email = j.ramirez@gmail.com

password=[string] (Al menos 8 caracteres, con una mayúscula, minúscula y un número).
Ejemplo: password = swordfish

password=[string]
Ejemplo: password = swordfish

reppassword=[string]
Ejemplo: reppassword = swordfish

Respuesta de éxito <El usuario se registró> 
 
{ 
    'result' : 'OK',
	'username' : 'j.ramirez'
}
Respuesta de error <Contraseña no cumple con los requisitos> 
 
{ 
    'result' : 'FAIL PASS' 
}

<Contraseñas no coinciden> 
 
{ 
    'result' : 'FAIL REPASS' 
}

<Email ingresado ya registrado> 
 
{ 
    'result' : 'FAIL EMAIL' 
}

<El username ingresado ya registrado> 
 
{ 
    'result' : 'FAIL USER' 
}
