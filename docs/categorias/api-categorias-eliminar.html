DELETE api/categorias/eliminar

Permite eliminar una categoría existente. Requiere autenticación de administrador.

URL del recurso http://tiendaapirest.aws.af.cm/api/categorias/eliminar
Parámetros

Requeridos:
idCat=[integer]
Ejemplo: idCat = 5

nomCat=[string]
Ejemplo: nomCat = Animales

username=[string] (Username de administrador).
Ejemplo: username = someadmin

Respuesta de éxito <La eliminación se realizó> 
 
{ 
    'result' : 'Eliminacion exitosa' 
}
Respuesta de error <Datos no enviados> 
 
{ 
    'result' : 'ERROR DATOS' 
}

<El registro no existe> 
 
{ 
    'result' : 'ERROR REGISTRO NO EXISTE' 
}

<Administrador no logueado> 
 
{ 
    'result' : 'UNAUTHORIZED' 
}
