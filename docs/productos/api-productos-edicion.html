PUT api/productos/edicion

Permite editar un producto existente.

URL del recurso http://tiendaapirest.aws.af.cm/api/productos/edicion
Parámetros

Requeridos:
idcategoria=[integer]
Ejemplo: idcategoria = 5

nombre=[string]
Ejemplo: nombre = Brand Car

idProd=[integer]
Ejemplo: idProd = 20

descripcion=[string]
Ejemplo: descripcion = 4-door, FW Engine

codigo=[integer]
Ejemplo: codigo = 1120

precio=[integer]
Ejemplo: precio = 69000000

existencias=[integer]
Ejemplo: existencias = 120

username=[string] (Username de administrador).
Ejemplo: username = someadmin

Respuesta de éxito <La edición se realizó> 
 
{ 
    'result' : 'Edicion exitosa' 
}
Respuesta de error <Datos no enviados> 
 
{ 
    'result' : 'ERROR DATOS' 
}

<El registro no existe> 
 
{ 
    'result' : 'ERROR REGISTRO NO EXISTE' 
}

<Administrador no logueado> 
 
{ 
    'result' : 'UNAUTHORIZED' 
}
