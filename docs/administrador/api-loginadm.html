POST api/loginadm

Permite loguear a un administrador existente.

URL del recurso http://tiendaapirest.aws.af.cm/api/loginadm
Parámetros

Requeridos:
username=[string]
Ejemplo: username = johng

password=[string]
Ejemplo: password = memento

Respuesta de éxito <Autenticación exitosa> 
 
{ 
    'result' : 'OK',
	'username' : 'johng'
}
Respuesta de error <Contraseña incorrecta> 
 
{ 
    'result' : 'FAIL PASS' 
}

<El username ingresado no está registrado> 
 
{ 
    'result' : 'FAIL USER' 
}
