Categorías

Lista de categorías, creación, edición y eliminación.

RecursoDescripción
GET api/categorias

Devuelve las categorías actuales.
GET api/categorias/id

Devuelve el nombre de la categoría, su id y sus productos, además de las categorías actuales. Requiere autenticación de administrador.
POST api/categorias

Permite registrar una nueva categoría. Requiere autenticación de administrador.
PUT api/categorias

Permite editar una categoría existente. Requiere autenticación de administrador.
DELETE api/categorias

Permite eliminar una categoría existente. Requiere autenticación de administrador.

Productos

Lista de productos, creación, edición y eliminación.

RecursoDescripción
GET api/productos

Devuelve las categorías y productos actuales.
POST api/productos

Permite registrar un nuevo producto. Requiere autenticación de administrador.
PUT api/productos

Permite editar un producto existente. Requiere autenticación de administrador.
DELETE api/productos

Permite eliminar un producto existente. Requiere autenticación de administrador.

Usuarios

Registro y login de usuarios.

RecursoDescripción
POST api/signup

Permite registrar un nuevo usuario.
POST api/login

Permite loguear a un usuario existente.

Canasta

Recuperación y manejo de canasta.

RecursoDescripción
GET api/canasta

Devuelve los productos agregados a la canasta por un usario.
POST api/canasta

Agrega productos a la canasta para un usuario.
PUT api/canasta

Permite realizar compra y actualizar existencias.
DELETE api/canasta

Permite eliminar todos los productos agragados a la canasta por un usuario.

Administrador

Login de administradores.

RecursoDescripción
POST api/loginadm

Permite loguear a un administrador existente.
