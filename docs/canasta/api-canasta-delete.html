DELETE api/canasta/delete

Permite eliminar todos los productos agragados a la canasta por un usuario.

URL del recurso http://tiendaapirest.aws.af.cm/api/canasta/delete
Parámetros

Requeridos:
username=[string]
Ejemplo: username = "Nombre de usuario"

Respuesta de éxito <La eliminación se realizó> 
 
{ 
    'result' : 'OK' 
}
Respuesta de error <La eliminación no se realizó> 
 
{ 
    'result' : 'FAIL' 
}
